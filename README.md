# 🎨 WawaKalu 🌟

**Aplicación educativa e interactiva para niñas y niños**

---

## 📖 Descripción

WawaKalu es un proyecto social colaborativo que busca crear una aplicación web educativa, divertida e inclusiva para niñas y niños. A través de actividades interactivas, los niños pueden aprender conceptos básicos mientras se divierten.

## ✨ Características

### Actividades Disponibles

🎨 **Colorea** - Dibuja y pinta con una paleta de colores vibrantes
- Canvas interactivo para dibujo libre
- Múltiples colores disponibles
- Fácil de usar en dispositivos táctiles

🔢 **Números** - Aprende a contar del 0 al 10
- Visualización grande y clara
- Nombres de números en español
- Navegación intuitiva

🔤 **ABC** - Explora el alfabeto español
- Incluye la letra Ñ
- Ejemplos de palabras para cada letra
- Diseño colorido y atractivo

⭐ **Formas** - Descubre formas geométricas básicas
- Círculos, cuadrados, triángulos y más
- Descripciones educativas
- Emojis para mejor comprensión

## 🚀 Cómo Usar

1. Abre `index.html` en tu navegador web
2. Selecciona una actividad haciendo clic en cualquier tarjeta
3. ¡Disfruta aprendiendo y jugando!
4. Usa el botón "Volver" para cambiar de actividad

## 💻 Tecnologías
